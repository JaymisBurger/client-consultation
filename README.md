# Client Consultation

A standalone prototype for salon and school client consultation forms.

## What it includes

- Form setup screen for creating and editing consultation form templates
- Interactive form preview
- Appointment intake screen for clients completing required forms
- Saved-progress and review flows for previously completed forms

## Run locally

```sh
npm start
```

Then open `http://127.0.0.1:4174`.

## Test

```sh
npm test
npm run build
```
